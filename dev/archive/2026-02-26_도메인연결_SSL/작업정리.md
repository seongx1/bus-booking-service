## 3. 가비아 DNS 설정

- **위치:** https://dns.gabia.com (또는 domain.gabia.com → DNS 관리)
- **도메인:** koreabuscharter.com 선택 → DNS 설정 → 레코드 수정
- **추가한 A 레코드:**

| 타입 | 호스트 | 값/위치       | TTL  |
|------|--------|---------------|------|
| A    | @      | 68.183.180.33 | 600  |
| A    | www    | 68.183.180.33 | 600  |

- **68.183.180.33** = Cloudways 서버(1574100) 공용 IP
- DNS 전파 후 `koreabuscharter.com`, `www.koreabuscharter.com` 이 해당 서버로 연결됨

---

## 4. SSL 인증서 설치 (HTTPS)

- **위치:** Cloudways → 앱 Korea bus charter → **SSL Certificate**
- **방법:** Let's Encrypt (무료)
- **진행한 작업:**
  1. **Let's Encrypt** 선택
  2. **Email Address:** 만료 알림 수신용 이메일 입력
  3. **Domain Name:** `koreabuscharter.com` 입력
  4. **+ Add Domain** → `www.koreabuscharter.com` 추가
  5. **Install Certificate** 클릭
  6. "Please wait while Let's Encrypt certificate installs..." 대기 (약 1~2분)
- **결과:** https://koreabuscharter.com, https://www.koreabuscharter.com 사용 가능
